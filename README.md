# RunMachine

## Setup:

-   Using esp now to connect with main body.
-   Using TFT screen 2.8 inch (240x320) and show logo when startup
-   Main body using ESP8266 F, tracker using ESP8266 F
-   Using HC595 in main body to show 6 LED
 
## TFT
